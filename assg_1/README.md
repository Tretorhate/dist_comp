# RPC Lab - Simple Remote Procedure Call

## How to Run

0. Based on assignment choose the proper folder to run
   ```bash
   cd assg_[number_of_assignment]
   ```
1. On server instance:
   ```bash
   python3 server.py
   ```
2. On client instance:
   ```bash
   python3 client.py [public_server_ip]
   ```
